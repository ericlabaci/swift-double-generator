# DoubleGenerator

Generate Swift doubles from protocols.

Written in Swift.

# Instalation
## Brew
`brew install ericlabaci/formulae/swift-double-generator`

## Build from source
Clone the repo and run
`make instal`

# Usage
`double-generator generate-double <file_path>`
